HOW TO PLAY

Place mirrors ( / and \ ) on the grid to redirect light beams from the emitters to the receptors.

Toggle between SUN and MOON mode — each level has one SUN receptor and one MOON receptor. Both must be lit simultaneously to win.

Click an empty cell to cycle: empty → / → \ → empty.
Right-click a mirror to remove it.

Use the EDITOR to create your own levels by placing walls, emitters, and receptors on the palette, then clicking the grid. Export as JSON.

Keyboard

T = Toggle mode  R = Reset  Esc = Close terminal/editor
